A GUI-based restaurant billing system built using Python and Tkinter.  
The application allows users to select food and drinks, calculate totals, apply tax, and generate a receipt.

## Features

- Food and drink selection
- Quantity-based billing
- Automatic tax calculation (15%)
- Service charge
- Subtotal and total calculation
- Receipt generation with bill number
- Reset and Exit options
- Built-in calculator

## Technologies Used

- Python 3
- Tkinter

1. Clone the repository:
   ```
   git clone https://github.com/your-username/repository-name.git
   ```

2. Navigate to the folder:
   ```
   cd repository-name
   ```

3. Run the program:
   ```
   python main.py
   ```
